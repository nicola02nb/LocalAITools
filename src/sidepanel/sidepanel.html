Gemini Nano AI Tools


A Chrome extension to chat with the Nano AI Api built into Chrome.

All models are run locally on your hardware using the chrome experiments that should be enabled:

  • Opimization on device model (runs model on local hardware)
  • LanguageModel
  • Summarizer
  • Translator
  • Language Detector
  • Writer
  • Rewriter

After you have enabled the models you desire, you have to restart your browser; and then, after the background download has finished, you can acces the models using the new tabs.
